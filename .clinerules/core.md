Always prioritize /ai_state/ files over chat history.

## Execution Discipline
Only work from TASK_QUEUE.md.

## No Architecture Reset
Never redesign working systems unless explicitly requested.

## Incremental Work Only
Make small, safe changes.

## Pre-Action Requirement

1. Read PROJECT_CONTEXT.md
2. Read AGENT_HANDOFF.md
3. Read TASK_QUEUE.md

## Post-Action Requirement
After completing tasks:
- update TASK_QUEUE.md
- update AGENT_HANDOFF.md
